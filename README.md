# 🐾 PetShop Mobile App

PetShop is a mobile application built using **React Native and Expo**.
The app provides a simple interface to explore pet-related products.

This project was created using **Expo** and uses **file-based routing with Expo Router**.

---

# 📱 Features

* Browse pet products
* View product information
* Simple and clean mobile UI
* File-based routing using Expo Router
* Cross-platform mobile support

---

# 🛠 Tech Stack

* React Native
* Expo
* TypeScript
* Expo Router

Main libraries used in this project:

# ⚙️ Setup Instructions

Follow these steps to run the project locally.

## 1️⃣ Clone the repository

```bash
git clone https://github.com/peermohamed07/PetShop.git
```

## 2️⃣ Navigate to the project directory

```bash
cd PetShop
```

## 3️⃣ Install dependencies

```bash
npm install
```

## 4️⃣ Start the application

```bash
npx expo start
```

After running the command, a QR code will appear.

You can open the app using:

* Expo Go (recommended)
* Android Emulator
* iOS Simulator

Scan the QR code using the **Expo Go app** on your mobile device.
